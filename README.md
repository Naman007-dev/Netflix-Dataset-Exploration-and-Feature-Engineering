# Netflix Content Analysis and Data Preprocessing using Python

## 📌 Overview
This project analyzes Netflix Movies and TV Shows data using Python. It covers data cleaning, preprocessing, feature engineering, encoding, scaling, and exploratory data analysis to uncover trends and patterns in Netflix content.

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## 🔹 Data Preprocessing
- Handled missing values
- Removed duplicates
- Standardized column names
- Extracted `year_added` from `date_added`
- Applied Label Encoding and One-Hot Encoding
- Performed Standardization and Min-Max Normalization

## 📊 Visualizations
- Histogram
- Boxplot
- Scatter Plot
- Correlation Heatmap

## 📈 Key Insights
- Most Netflix content was released after 2000.
- Release years are heavily skewed toward recent years.
- Older titles appear as outliers.
- Duration and release year show a weak negative correlation.
